# Deploy to GitHub Pages

## Steps to deploy:

1. **Create GitHub Repository:**
   - Go to github.com
   - Click "New repository"
   - Name: `gagan-portfolio`
   - Make it public
   - Don't initialize with README

2. **Upload Files:**
   - Drag and drop all files from `C:\gagan-portfolio\` to the repository
   - Or use GitHub Desktop/VS Code

3. **Enable GitHub Pages:**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Source: Deploy from branch
   - Branch: main
   - Folder: / (root)
   - Click Save

4. **Your website will be live at:**
   `https://yourusername.github.io/gagan-portfolio`

## Files to upload:
- index.html
- css/style.css
- js/script.js
- README.md
- images/ (folder - if you add local images later)

The website will be live within 5-10 minutes after enabling GitHub Pages.
